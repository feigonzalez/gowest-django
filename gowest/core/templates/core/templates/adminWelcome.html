Bienvenido/a a la página de administración

Al costadoEn la barra superior se encuentran las pestañas de administración:

  • La pestaña Cuenta le permite actualizar sus datos o cambiar sus parámetros de seguridad.
  • La pestaña Productos muestra una lista de los productos que vende la tienda.
  • La pestaña Categorías muestra una lista de las categorías a las que cada producto puede pertenecer.
  • La pestaña Clientes muestra una lista de los clientes que se han registrado en la tienda.
  • La pestaña Ventas muestra una lista de las ventas realizadas.
  • La pestaña Administradores muestra una lista de los adminstradores que tienen acceso a esta página.
